# Python製イーサネットとWi-Fi自動切り替えGUIアプリ計画書

## 1. 背景
Windows PCにイーサネットとWi-Fiが接続されている場合、イーサネットが常に優先されてしまう問題がある。この問題を解決するために、イーサネットとWi-Fiの自動切り替えを行うGUIアプリを作成する。実際には自動切り替えというよりは片方のアダプターを無効化して一方を有効化というのを排他的に切り替えられるアプリである。

## 2. 目的
イーサネットを無効化すると、Wi-Fiが自動的に有効化されるようにする。
Wi-Fiを無効化すると、イーサネットが自動的に有効化されるようにする。

## 3. 要件
- Python 3.10以降
- tkinter

## 4. 機能
- イーサネットアダプターとWi-Fiアダプターを自動補足する
- イーサネットアダプターを無効化すると、Wi-Fiアダプターが有効化される
- Wi-Fiアダプターを無効化すると、イーサネットアダプターが有効化される
- イーサネットアダプターとWi-Fiアダプターの状態を表示する
